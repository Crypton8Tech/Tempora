Как настроить Stripe

  1. Зарегистрируйтесь на stripe.com
  2. Перейдите в Developers → API keys
  3. Скопируйте Publishable key и Secret key
  4. Для получения Webhook Secret создайте webhook в Developers → Webhooks, указав URL: https://ваш-домен/api/stripe/webhook
  5. Выберите событие checkout.session.completed
